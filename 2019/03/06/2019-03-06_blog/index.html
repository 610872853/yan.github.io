记录个人博客主题修改

发表于 2019-03-06 | 分类于 博客搭建


  本博客主要记录个人定制博客主题的过程，主要采用了简洁风格的NexT主题。在Next主题的基础上，添加自定义头像、修改标题背景和添加点击爆炸效果。

一、 采用NexT主题

  1. 安装NexT，点击Git Base Here，打开git命令行窗口，输入命令：
    git clone https://github.com/iissnan/hexo-theme-next themes/next

  2. 设定主题，打开/themes/next/_config.yml
  3. 设置菜单
  4. 设置侧边栏位置

二、 添加自定义头像

  1. 添加头像链接，打开/themes/next/_config.yml
  2. 添加圆角和旋转样式，打themes\next\source\css\_common\components\sidebar\sidebar-author.styl，修改头像样式
